class controls.TextField(Widget)

A TextField provides a simple single-line text entry area. Typed characters are inserted into the text, and characters are deleted using Delete or Backspace. Clicking on the text field gives it the keyboard focus and sets the insertion point.

There is currently no support for selection, copy/paste or tabbing between text fields.

Constructor

TextField(width, font = None)
The width can be either an integer or a string. If an integer, it specifies the width in pixels; if a string, the widget is made just wide enough to contain the given text. The height is determined by the height of a line of text in the given font, or the default font from the theme.

Theme Properties

fg_color
Colour in which to display the text and insertion bar.

font
Font in which to display the text.

Attributes

upper
If true, text typed into the field is forced to upper case.
text
The current text, provided the get_text and set_text methods have not been overridden.
insertion_point
The current position of the insertion point. May be set to None to position it at the end of the text.
enter_action
A function of no arguments to be called when Return or Enter is pressed. If not specified, Return and Enter key events are passed to the parent widget.

escape_action
A function of no arguments to be called when Escape is pressed. If not specified, Escape key events are passed to the parent widget.

Methods

get_value()
set_value(text)
These methods get and set the text being edited. They are provided for interface compatibility with other value-editing widgets.

Abstract Methods

get_text()
set_text(text)
Internally, the widget uses these methods to access the text being edited. By default they access the text attribute, but by overriding them you can arrange for the widget to edit text being held somewhere else.
