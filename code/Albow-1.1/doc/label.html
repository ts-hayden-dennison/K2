class controls.Label(Widget)

A Label displays a piece of static text in a chosen font and colour.

Constructor

Label(text, width = None, font = None)
Initializes the label with the given text and font. If a width is specified, it is used, otherwise the label is initially made just wide enough to contain the text. The text may consist of more than one line, separated by '\n'. The initial height is determined by the number of lines and the font specified or the default font from the theme.

Theme Properties

font
The font in which to display the text.

fg_color
The colour in which to display the text.

Attributes

text
The text to be displayed. This can be changed dynamically, but the label won't automatically resize to accommodate the new text.
